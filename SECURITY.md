# Security Policy

## Supported Versions

We support the following versions of AI-SKILL:

| Version | Supported          |
| ------- | ------------------ |
| 4.x.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within AI-SKILL, please send an e-mail to the maintainers. All security vulnerabilities will be promptly addressed.

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any other relevant information

You can expect a response within 48 hours, and a fix within 7-14 days depending on severity.

## Security Updates

We will notify users of critical security updates via:
- GitHub Security Advisories
- Project README updates
- Changelog entries

## Best Practices for Using Skills

When using skills from this repository:
- Review the skill code before using it
- Be cautious with skills that require elevated permissions
- Keep your skills updated to the latest versions
- Report any suspicious behavior

## Contact

For any security concerns, please reach out via GitHub Issues or the repository owner.
