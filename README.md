## Overview

AI-SKILL is the largest universal agent skill library, with **2677+ production-ready skills** compatible with all major platforms: Claude Code, Claude.ai, Cursor, Gemini CLI, Codex CLI, Doubao, Qwen, Spark, Moonshot, Baidu, Alibaba.

**Important Note:** This project is a curated collection and modification of existing open-source skills. All skills are from the original authors listed below, standardized and enhanced for cross-platform compatibility.

## Quick Start

### Clone and Use

```bash
git clone https://github.com/badhope/AI-SKILL.git
cd AI-SKILL
```

### Configure Your Platform

1. **Claude Code**: Configure in `~/.claude/plugins.json`
2. **Cursor**: Copy skills to your project or configure rules
3. **Gemini CLI**: Follow Gemini's skill configuration

### Basic Usage

All skills use a standardized format:

```yaml
---
name: skill-name
version: 1.0.0
description: Brief description
author: Original Author Name
category: development
tags: [tag1, tag2]
license: MIT
platforms: [claude, cursor, gemini, doubao, qwen, ...]
---

# Skill Name

## Overview
Describe what this skill does

## When to Use
- List clear usage scenarios

## How to Use
Step-by-step instructions
```
